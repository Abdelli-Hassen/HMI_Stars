\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}
\begin{spacing}{1.5}
This graduation project focuses on the design and implementation of a real-time communication and administrative management ecosystem named \textbf{HMI Stars}. Built specifically for \textbf{HMI Stars Consulting}, this ecosystem aims to simplify, automate, and centralize social, administrative, and financial exchanges between the enterprise and its corporate clients. The system architecture is established on a Flutter monorepo containing two distinct applications connected to a single \textbf{Supabase} backend: a web-based administration platform for the enterprise's managers, and a cross-platform mobile application for client managers.

The web administration platform enables managers to supervise affiliated companies, collect accounting files, schedule urgent tasks, issue disciplinary warning letters, and centralize messaging histories. The mobile application allows clients to track their business status, declare employee absences and leaves, submit daily attendance tracking, upload administrative documents, and chat instantly with the enterprise. This ecosystem integrates advanced security mechanisms via PostgreSQL Row Level Security (RLS) policies and ensures authentication flow consistency on the web by implementing an adaptive routing protocol combined with Supabase's implicit auth flow.

\textbf{Keywords:} Flutter, Supabase, PostgreSQL, Real-time, Enterprise Messaging, Administrative Management, RLS.
\end{spacing}
