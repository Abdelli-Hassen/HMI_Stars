\chapter*{General Conclusion}
\addcontentsline{toc}{chapter}{General Conclusion}
This graduation project involved designing and implementing the real-time application ecosystem \textbf{HMI Stars} for the consulting enterprise \textbf{HMI Stars Consulting}. This work successfully addressed a real-world requirement: improving administrative, tax, and HR exchanges between an agency and its corporate clients.

The objective of centralizing communication and ensuring the traceability of documents and employee records was achieved. On the technical side, the Flutter framework demonstrated high efficiency in cross-platform development, reducing development overhead by sharing code logic. Similarly, Supabase provided a reactive and secure backend, with PostgreSQL Row Level Security (RLS) policies guaranteeing data isolation.

Key personal contributions in this project include structuring the relational data model, setting up global state management, and resolving complex web routing and session recovery issues.

Nonetheless, future enhancements are planned. In the short term, we plan to implement push notifications (via Firebase Cloud Messaging) to alert client managers about document updates or urgent tasks. In the long term, adding automated PDF payslip generation based on approved monthly attendance logs could expand HMI Stars into a full-scale payroll tool.

This internship has been a highly valuable professional and personal experience, allowing me to strengthen my software engineering skills while addressing real-world security and deployment constraints.
